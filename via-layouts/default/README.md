## Default Layout

 ### Layer 0 (Default)
| Position | Default Function |
|----------|------------------|
| Key 1 | Page Up |
| Key 2 | Page Down |
| Key 3-8 | F1, F2, F3, F4, F5, F6 |
| Encoder 1 | Volume Up/Down |
| Encoder 1 Press | Mute |
| Encoder 2 | RGB Brightness Up/Down |
| Encoder 2 Hold | Enable next layer |
