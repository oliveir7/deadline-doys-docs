## App Hotkeys

| Command(s) | App(s) | What it does |
| --- | --- | --- |
| `Ctrl+Shift+I` | AMD Adrenaline | Take screenshot |
| `Ctrl+Shift+E` | AMD Adrenaline | Record instant replay |
| `Ctrl+Shift+S` | AMD Adrenaline | Toggle recording |
| `Alt+Shift+G` | AMD Adrenaline | Activate 2x frame gen |
| `Ctrl+Shift+O` | AMD Adrenaline | Toggle overlay |
| `Alt+F1` | NVIDIA App | Take screenshot |
| `Alt+F10` | NVIDIA App | Record instant replay |
| `Alt+F3` | NVIDIA App | Open filters (RTX HDR) |
| `Alt+Shift+F10` | NVIDIA App | Toggle instant replay |
| `Alt+R` | NVIDIA App | Toggle recording |


## Navigation

| Macro | What it does |
| --- | --- |
| `Page Up` | Moves up one page |
| `Page Down` | Moves down one page |
| `F1` - `F6` | Sends function keys F1 through F6 |
| `KC_MYCM` | Opens My Computer |
| `KC_WWW_SEARCH` | Opens search |

## Media and Audio

| Macro | What it does |
| --- | --- |
| `KC_MPLY` | Toggles play / pause |
| `KC_VOLU` | Raises volume |
| `KC_VOLD` | Lowers volume |
| `Mute` | Mutes audio |
| `Volume Up/Down` | Adjusts volume from the encoder |

## Capture and Recording

| Macro | What it does |
| --- | --- |
| `Screenshot` | Captures a screenshot |
| `Record instant replay` | Saves the recent replay buffer |
| `Toggle recording` | Starts or stops recording |
| `Toggle instant replay` | Enables or disables instant replay |
| `Activate 2x frame gen` | Turns on 2x frame generation |
| `Open filters (RTX HDR)` | Opens the RTX HDR filters panel |
| `Toggle overlay` | Shows or hides the overlay |
| `Toggle performance stats` | Shows or hides performance stats |

## Lighting and RGB

| Macro | What it does |
| --- | --- |
| `RGB Brightness Up/Down` | Changes RGB brightness with the encoder |
| `RGB_VAD` | Lowers RGB brightness |
| `RGB_VAI` | Raises RGB brightness |
| `RGB mode prev` | Switches to the previous RGB mode |
| `RGB mode next` | Switches to the next RGB mode |
| `Toggle RGBs` | Turns RGB lighting on or off |
| `Change color / hue +` | Increases hue |
| `Backlight Brightness` | Changes backlight brightness |

## Layers

| Macro | What it does |
| --- | --- |
| `Enable next layer` | Activates the next layer when held |
| `Activate next layer` | Activates the next layer with the bottom encoder press |

## Bluetooth and Connectivity

| Macro | What it does |
| --- | --- |
| `BLE_1` | Switches to Bluetooth channel 1 |
| `BLE_2` | Switches to Bluetooth channel 2 |
| `BLE_OFF` | Disconnects Bluetooth |
| `Bluetooth connection 1` | Starts Bluetooth pairing slot 1 |
| `Bluetooth connection 2` | Starts Bluetooth pairing slot 2 |
| `Disconnect bluetooth` | Disconnects Bluetooth |

## System and OS Shortcuts

| Macro | What it does |
| --- | --- |
| `WinLock` | Toggles Windows key lock |
| `CG_TOG` | Swaps Ctrl and Gui |
| `Win+Alt+B` | Toggles HDR in Windows 11 |
| `Mission Control` | Opens macOS Mission Control |
| `Launch Pad` | Opens macOS Launch Pad |
| `TRGB_TOG` | Toggles top RGB |
| `BAT_SHOW` | Shows battery level light |
| `USB_TOG` | Toggles USB output |
| `Battery level` | Displays battery level |

## Physical Controls

| Macro | What it does |
| --- | --- |
| `Power off device` | Powers off the device |
| `Toggle backlight` | Toggles backlight power on USB mode |
| `Disable Bluetooth` | Turns Bluetooth off |
